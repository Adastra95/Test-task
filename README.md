Небольшое веб-приложение для создания и отображения статей.
Все данные сохраняются в LocalStorage, поэтому проект работает полностью на стороне клиента и не требует сервера.

Возможности

1.Создание статьи с заголовком и текстом

2.Сохранение всех данных в LocalStorage

3.Автоматический вывод всех статей на главной странице

4.Отображение даты создания и обновления

5.редактирование статей 

6.просмотр статьи

структура проекта 
/
├── index.html # Главная страница. Отображение всех статей
├── form_create_article.html # Страница создания статьи
├── css/
│ └── style.css # Основные стили
├── js/
│ ├── script.js # Сохранение статей в LocalStorage
│ └── searchLocalStorage.js # Поиск/получение данных из LocalStorage, редактирование и просмотр статьи   

Запуск проекта 

1.скачать live server в vs code 

2.клинкуть пкм по файлу index.html 

3.готово
